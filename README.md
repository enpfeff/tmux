# tmux

Personal tmux configuration with session management.

## Dependencies

- [fzf](https://github.com/junegunn/fzf) - fuzzy finder for session selection

## Features

- Configures tmux with custom keybindings and settings
- Allows you to quickly sort through and switch between tmux sessions using fzf

## Usage

Press `prefix + f` to open the session switcher, which uses fzf to let you fuzzy-find and switch between your project directories as tmux sessions.
